# Enterprise Homelab Infrastructure

Production-grade homelab demonstrating enterprise IT operations and automation.

## What I Built

Three virtual machines running on Proxmox with:
- Centralized identity management (FreeIPA)
- Automated user provisioning (Ansible)
- Monitoring and alerting (Prometheus/Grafana)
- SSL/TLS encryption for all services
- Secrets management (Ansible Vault)

## Technology Stack

- **Virtualization:** Proxmox
- **Operating Systems:** Rocky Linux, Ubuntu Server
- **Automation:** Ansible
- **Containers:** Docker, Docker Compose
- **Identity Management:** FreeIPA
- **Monitoring:** Prometheus, Grafana, Loki
- **Web Services:** Nginx, PostgreSQL, MariaDB

## Key Achievements

- Automated user onboarding (15-20 min → under 2 min)
- SSL/TLS across all services with internal CA
- Centralized secrets management
- Automated nightly backups
- Production-grade monitoring
